## ⚡ Key Interactions

### Analytics:
1. Select campaign → Fetch data
2. Change date filter → Refresh metrics
3. View platform bars → See percentages
4. Check leaderboard → See top clippers

### Connected Accounts:
1. Click "Connect" → Open modal
2. Fill form → Submit
3. View card → See connected account
4. Click "Disconnect" → Confirm → Remove

### Wallet:
1. Click "Add Funds" → Open modal
2. Enter amount → Submit → Update balance
3. Click "Withdraw" → Open modal
4. Enter amount → Validate → Submit
5. View history → See all transactions

## 🎯 User Flows

### Campaign Analytics Flow:
```
Login → Dashboard → Analytics
    ↓
Select Campaign
    ↓
Choose Date Range (Optional)
    ↓
View Metrics:
    • Budget vs Spent
    • Total Views/Likes/Shares
    • Platform Distribution
    • CPM
    • Featured Clippers
    ↓
Export (Future) / Share (Future)
```

### Connect Account Flow:
```
Login → Connected Accounts
    ↓
See Connected: YouTube ✓, Instagram ✓
    ↓
Available: TikTok
    ↓
Click "Connect TikTok"
    ↓
Enter: Username, Account ID
    ↓
Submit → OAuth (Future) → Connected!
    ↓
View All Platforms ✓✓✓
```

### Wallet Transaction Flow:
```
Login → Wallet
    ↓
View Balance: $5,000
    ↓
Option 1: Add Funds
    • Click "Add Funds"
    • Enter $100
    • Optional description
    • Submit → Balance: $5,100
    ↓
Option 2: Withdraw
    • Click "Withdraw"
    • Enter $50
    • Validate balance
    • Submit → Balance: $5,050
    ↓
View Transaction History
    • All transactions
    • Sorted by date
    • Color-coded
```
